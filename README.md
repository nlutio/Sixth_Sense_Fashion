Sixth Sense Fashion House 👗
A Collaborative Web Development Project – CSC 106

📝 Overview
Sixth Sense Fashion House is a conceptual digital boutique designed and developed by a team of 15 students. This project serves as an exploration of modern web development principles, focusing on modular structure, collaborative workflows, and the integration of diverse content components into a unified brand identity.

🏗️ Project Architecture
To ensure scalability and ease of maintenance, the project is structured as a series of independent but cohesive HTML pages tied together by a shared stylesheet and navigation structure.

🚀 Key Technical Features
Consistent Navigation: A unified site structure across all pages using shared components.

Component-Based Logic: A modular approach where the site is partitioned into functional groups (Forms, Tables, Content, and Assets).

Responsive Layout: Built with a focus on CSS variables for consistent branding and a "mobile-first" navigation approach.

Interactive DOM: Integration of JS features (marquee and validation logic) to enhance the user experience.

🛠️ Development Workflow
As the Project Lead, I managed the development lifecycle by:

Defining the DOM Skeleton: Establishing the core HTML structure to prevent integration conflicts.

Version Control: Coordinating the GitHub repository to merge contributions from 15 team members into a single, functional source of truth.

Design System: Implementing global CSS variables to ensure all component groups maintained brand consistency.
